## 📋 Funcionalidades  
✔ Guardar series, repeticiones y peso  
✔ Crear plantillas de entrenamiento  
✔ Sincronización con Firebase  

## 🔧 Tecnologías  
- Kotlin, Jetpack Compose  
- Firebase Auth, Firestore  
- MVVM, Coroutines  

## 🚀 Instalación  
1. Clonar el repo  
2. Abrir en Android Studio  
3. Ejecutar la app en un emulador
